# Harness V3.13 Download Guard

Date: 2026-05-26

Purpose: preserve V3.12's Writer/Impress/Chrome wins while restoring the repeated numbered PDF download behavior that V3.10 handled.

Key changes from V3.12:
- Version marker and metadata updated to V3.13.
- Kept V3.12 named-green guidance and V3.10 Office DONE gate.
- Added a narrow repeated numbered PDF download-set drift detector.
- After repeated download attempts in a week/lecture/slide/part PDF set, the runner rejects browser navigation, scraping scripts, and near-duplicate download actions unless the next action is one concise loop that downloads the full numbered set and verifies file names/sizes.

Expected effect:
- Preserve Writer, Impress, and Chrome.
- Reduce max-step download loops by forcing a final download-and-verify route.

Risk:
- The guard may reject a legitimate late recovery action if it does not look like a concise loop plus verification.
- PDF generation remains intentionally unchanged because V3.11's PDF guard caused broad regressions.
