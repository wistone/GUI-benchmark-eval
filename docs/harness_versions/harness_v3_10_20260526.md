# Harness V3.10 Office DONE Fix

Date: 2026-05-26

Purpose: preserve V3.9's wins while fixing a runner bug in the Office early-DONE gate.

Key changes from V3.9:
- Version marker and metadata updated to V3.10.
- Ctrl+S save detection now uses whitespace-tolerant matching, so `pyautogui.hotkey('ctrl', 's')` is recognized as a save.
- Exact-color palette-click rejection remains enabled.
- Broad Terminal/download rejection remains disabled.

Expected effect:
- Writer/Calc/Impress tasks that save and immediately return DONE should be forced into a post-save verification step.
- Impress exact-color and download behavior should remain close to V3.9.

Risk:
- Verification may still be insufficient if the model checks the wrong paragraph/object property.
